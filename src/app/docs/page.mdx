## Adding content

Create a `page.mdx` file inside a route directory under `src/app`. For example,
`src/app/guides/getting-started/page.mdx` becomes `/guides/getting-started/`.

Import a colocated module when a page needs custom styles:

```mdx
import styles from './page.module.css'

<aside className={styles.callout}>Scoped content</aside>
```

```tsx
export function Callout({ children }: { children: React.ReactNode }) {
  return <aside>{children}</aside>
}
```
